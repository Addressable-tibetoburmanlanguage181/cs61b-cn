# Lab05 - 自我反思（Self Reflection）

## 考察知识点
- 代码设计回顾与自我评估
- 与搭档的协作经验总结
- 对数据结构实现的反思

## 需要完成的任务

填写 `self_reflection.txt`，回答关于 **ArrayDeque61B** 和 **LinkedListDeque61B** 实现的反思问题：

### 每个数据结构需要回答的问题：
1. 工作方案的哪些思路可以让你的实现更好？
2. 你的代码中有什么让你感到自豪的？
3. 你的代码中有什么不够优雅的地方？
4. 你从搭档那里学到了什么？
5. 你希望在写代码之前知道什么？
6. 如果重新写，你会做什么不同的选择？

### Meta：
- 你喜欢这种设计评审过程吗？

## 如何测试
- 无自动测试
- 至少回答 4 个问题
