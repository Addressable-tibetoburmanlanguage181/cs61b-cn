# Proj3 - Bear Maps（地图路由）

## 考察知识点
- 图数据结构与最短路径算法（A* / Dijkstra）
- 地图数据解析（OpenStreetMap XML 格式）
- 空间数据结构（K-D Tree / Quad Tree）
- 图块渲染引擎（复用 Lab09 的 tileengine）
- 随机世界生成
- 文件 I/O 与 XML 解析
- 用户交互（键盘 + 鼠标）

## 需要完成的任务

### 在 `World.java` 中构建你自己的世界
- `World.java` 目前是空的，需要完全自行设计
- 使用 Lab09 学到的图块引擎技术

### 项目通常包含以下核心功能：
1. **地图数据加载** — 解析 OSM XML 格式的地图数据
2. **图构建** — 将路口和道路建模为图
3. **最短路径** — 实现 A* 或 Dijkstra 算法
4. **地图渲染** — 将地图绘制到屏幕上
5. **交互功能** — 缩放、平移、搜索地点、导航

## 如何测试
- 运行 `Main.java` 启动地图应用
- 通过鼠标和键盘交互测试功能
- 具体测试方式需参考课程网站上的详细说明
