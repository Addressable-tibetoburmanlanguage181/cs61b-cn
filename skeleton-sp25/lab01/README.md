# Lab01 - IntelliJ 与 JUnit 入门

## 考察知识点
- IntelliJ IDEA 项目结构与配置
- Java 类、方法、参数
- JUnit 5 测试框架基本使用
- Google Truth 断言库（`assertThat`）
- 使用 `StdIn` 读取用户输入

## 需要完成的任务
- **本实验主要是学习工具使用，代码已完整提供**
- 理解 `Arithmetic.java` 中 `product` 和 `sum` 方法的实现
- 学会在 IntelliJ 中运行测试（右键测试文件 → Run）
- 学会使用终端运行程序（通过 `main` 方法交互式输入）

## 如何测试
1. 运行 `ArithmeticTest.java` — 验证 product 和 sum 方法的正确性
2. 运行 `Arithmetic.java` 的 main 方法 — 交互式输入两个整数，查看乘积与和
